# Font Sources

The font files in this directory were downloaded from the official Google Fonts repository:

- Inconsolata: https://github.com/google/fonts/tree/main/ofl/inconsolata
- Source Sans 3: https://github.com/google/fonts/tree/main/ofl/sourcesans3

License files are stored next to the fonts:

- `Inconsolata-OFL.txt`
- `SourceSans3-OFL.txt`

The Inconsolata `FONTLOG.txt` is also included as `Inconsolata-FONTLOG.txt`.
